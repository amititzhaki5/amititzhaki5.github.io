Travel Planning Application

Designed and deployed an Android application that creates a travel itinerary for a user based on their destination, travel dates, and length of stay.

The application generates a curated list of activites to do and places to visit in the city that the user is traveling to. The app has an embedded Google Maps features that allows the user to see the location of the city they are visiting, as well as the location of each place listed on the itinerary.

AI De-Duplication

Rearched and tested large language models to determine how they could be used in the detection of duplicate code within large code bases.

The research was mainly conducted using Hugging Face, an online platform for the AI (artificial intelligence) and ML (machine learning) community. Hugging Face features a runtime environment that allows users to test models right on their site. My research mainly consisted of testing different models and focusing on the accuracy, reliability, and reproducibility of responses from the models. The motive for this research was to determine if AI models could be used for a task such as the de-deuplication of code. De-duplication is necessary for both increase code efficiency and maximum storage availability.

Ping Pong Game

Designed and deployed an Android application that resembles the game of ping pong.

The application begins with a stationary ball positioned at the top of the screen, and a paddle at the bottom. The user can drag their finger across the screen to move the paddle left or right. Once the user touches the screen, the ball falls and the user must hit it with the paddle. If the user strikes the ball, a sound plays and the ball bounces off the paddle. The ball will then hit one of the walls and eventually come back down. Then, the user will have to hit the ball again. If the user misses the ball, the game is over. The app keeps track of how many times the user hits the ball and stores the user's highest score. If the user closes and reopens the app, their high score will not be lost.
